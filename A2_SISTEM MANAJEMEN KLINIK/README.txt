Kode Klinik

1. Import `klinik.sql` ke PostgreSQL.
2. Jalankan PostgreSQL lokal.
3. Atur kredensial database lewat:
   - environment variable `DB_URL`
   - environment variable `DB_USER`
   - environment variable `DB_PASSWORD`
   atau ubah nilai default di `db/DBConnection.java`
4. Jalankan dari IDE atau:
   - Windows: `run.bat`
   - Linux/macOS: `run.sh`

Main class:
- `KodeKlinik`
